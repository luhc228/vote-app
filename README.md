## Requirements

- Node.js >= 22
- pnpm >= 9
- [MongoDB Atlas](https://www.mongodb.com/docs/atlas/)

Don't forget set `process.env.MONGODB_URI=your-MONGODB-URI` which you can find in MongoDB Atlas dashboard.
